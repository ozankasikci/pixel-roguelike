## Decisions Made

- `useModelOverride=true` for all CathedralScene mesh entities because all model matrices are pre-computed with arbitrary transform chains (translate+rotate+scale sequences that cannot be decomposed back to euler angles)
- GLM `vec3(scalar)` constructor is explicit in newer GLM -- brace-initialized `{1.0f}` in aggregate constructors causes compile error; must use `glm::vec3(1.0f)` explicitly
- SceneManager lives in main() for now -- Application could own it in a future phase but keeping it explicit simplifies this restructure
